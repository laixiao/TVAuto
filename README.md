# TV Auto

TV Auto 是一款使用 WebView 的轻量级电视直播软件。操作超级简单，将网络视频直播资源整合为频道列表，在 Android TV、手机和平板上提供一致的沉浸式播放体验。

---

## 功能概览

- 支持手动添加或批量导入频道，只需提供名称和 URL
- 理论兼容大部分 16:9 视频网页
- 支持遥控器和触摸屏操作
---

## 使用须知
- 默认频道列表为示例用途，不保证可用性与合法性
- 请用户自行核实、添加合规的频道
- 使用过程中请遵守当地法律法规，开发者不承担任何法律责任
---

## 快速使用

- 单个添加：输入频道名称和直播地址
- 批量导入：格式如下（以英文分号结尾）

  ```
  "频道名称","频道地址";
  ```

  示例：

  ```
  "CCTV-1 综合","https://tv.cctv.com/live/cctv1/";"江苏卫视","https://live.jstv.com/";
  ```
